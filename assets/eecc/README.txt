EECC-/Pflichtunterlagen für das Auftragsformular
=================================================

Legen Sie die folgenden PDF-Dateien in DIESEN Ordner (assets/eecc/).
Sobald eine Datei vorhanden ist, wird sie im Formular automatisch als
"Öffnen / Herunterladen" verlinkt. Fehlt eine Datei, erscheint der Eintrag
als "folgt vor Vertragsschluss" – die Erhalt-Bestätigung bleibt Pflicht.

Erwartete Dateinamen (exakt so benennen):

  vertragszusammenfassung.pdf   – EU-Pflichtformular (§ 55 TKG)
  leistungsbeschreibung.pdf     – Leistungsbeschreibung / Produktinformationsblatt
  preisliste.pdf                – Preisliste
  agb.pdf                       – Allgemeine Geschäftsbedingungen
  widerrufsbelehrung.pdf        – Widerrufsbelehrung + Muster-Widerrufsformular

Die Datenschutzinformationen verlinken auf datenschutz.html (bereits vorhanden).

Namen ändern? Dann in js/formular.js im Block CONFIG.documents die "file"-Pfade
anpassen. Dort können Sie auch Unterlagen ergänzen oder entfernen.
